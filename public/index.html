API Películas y Series

Gestioná, añadí y organizá tus películas y series favoritas con esta API

  • GET /api/peliculas - Listar todas las películas
  • POST /api/peliculas - Crear nueva película 
                                {
                                "titulo": "Nombre de la película",
                                "descripcion": "Descripción",
                                "director": "Nombre del director",
                                "anio": 2025,
                                "genero": "Acción",
                                "favorita": false
                            }
  • GET /api/peliculas/:id - Obtener película por ID
  • PUT /api/peliculas/:id - Actualizar película
  • DELETE /api/peliculas/:id - Eliminar película
  • GET /api/series - Listar todas las series
  • POST /api/series - Crear nueva serie 
                                {
                                "titulo": "Nombre de la serie",
                                "descripcion": "Descripción",
                                "director": "Nombre del director",
                                "anio": 2025,
                                "genero": "Drama",
                                "temporadas": 1,
                                "favorita": false
                            }
  • GET /api/series/:id - Obtener serie por ID
  • PUT /api/series/:id - Actualizar serie
  • DELETE /api/series/:id - Eliminar serie
  • GET /api/users - Listar usuarios
  • POST /api/users - Crear usuario
